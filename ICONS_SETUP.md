# App Icons Setup

The app requires the following icon files in the `public/` directory:

1. **icon-192x192.png** - 192x192 pixels
2. **icon-512x512.png** - 512x512 pixels
3. **favicon.ico** - Standard favicon

## Design Guidelines

- **Background:** Pure black (#000000)
- **Accent:** Neon green (#00FF41)
- **Symbol:** Terminal cursor ">" or command prompt
- **Style:** Minimal, cyberpunk, hacker aesthetic

## Quick Creation Options

### Option 1: Use an Online Tool
- Visit [favicon.io](https://favicon.io/favicon-generator/)
- Use black background with green text
- Generate and download

### Option 2: Design Your Own
- Create a 512x512 PNG with your design
- Use tools like Figma, Photoshop, or GIMP
- Export as PNG
- Resize to 192x192 for the smaller version

### Option 3: Use Placeholder
For testing, you can use simple colored squares:
- Create solid black squares with a green ">" symbol
- Use any image editor

## Installation

Once you have your icons:
1. Place them in the `public/` folder
2. Ensure they're named exactly as listed above
3. The app will automatically use them
